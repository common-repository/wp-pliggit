== Description ==

Aggiunge un bottone ai vostri articoli per proporli e votarli su Pligg.it. Automatico e facilissimo, consente di decidere in quale posizione far apparire il bottone di Pligg.it nel vostro blog.

== Installation ==

METODO A: Il modo più semplice e veloce
1. Caricate `wp-pliggit.php` nella cartella `/wp-content/plugins/`
2. Attivate il plugin tramite il menù 'Plugins' in WordPress
3. Fatto!  Il bottone Pligg.it apparirà nei vostri articoli automagicamente!
4. Andando poi in "Opzioni" ---> "Opzioni Pligg.it" potrete personalizzare la posizione in cui il bottone apparirà.

METODO B: Inserire il bottone Pligg.it manualmente solo in qualche articolo
1. Caricate `wp-pliggit.php` nella cartella `/wp-content/plugins/`
2. Attivate il plugin tramite il menù 'Plugins' in WordPress
3. Inserite `<!--pliggit-->` in un punto qualsiasi dell'articolo dove volete posizionare il bottone. Effettuare questa operazione utilizzando l'editor di codice, non quello visuale.
4. In "Opzioni" ---> "Opzioni Pligg.it" selezionare "Tramite Inserimento Codice" e salvare. 

METODO C:  Inserire il bottone Pligg.it automaticamente in tutti gli articoli in una posizione particolare all'interno della "loop"
1. Caricate `wp-pliggit.php` nella cartella `/wp-content/plugins/`
2. Attivate il plugin tramite il menù 'Plugins' in WordPress
3. Inserite `<?php show_pliggit(); ?>` in un punto qualsiasi del tema dove volete posizionare il bottone, all'interno della Wordpress "loop".
4. In "Opzioni" ---> "Opzioni Pligg.it" selezionare "Tramite Inserimento Codice" e salvare.
